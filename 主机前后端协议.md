### 1、开/关机

#### 前端 -> 主机

参数：

| 名称  | 类型 |       备注       |
| :---: | :--: | :--------------: |

```javascript

{
 "event_id": 11,

|  enddata  | string |     报表查询的结束时间     |

 } 

}
```
